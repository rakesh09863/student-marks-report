# student-marks-report
Features
Validates student number

Validates student name (only alphabets allowed)

Validates marks for each subject (0–100)

# Calculates:

Total marks

Percentage

Grade

Displays a clean formatted report

# Subjects Used

CM

CPPM

OSM

Each subject carries 100 marks, totaling 300 marks.

 # Grade Criteria
Total Marks	Grade
250–300	Distinction
200–249	First
150–199	Second
120–149	Third
Below 120	Fail
Any subject < 40  Fail 
 
# How It Works

Enter student number

Enter student name (only alphabets allowed)

Enter marks for each subject

Program calculates total, percentage, and grade

Displays a formatted marks report
